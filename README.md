# Algorithm Playground

## Overview

Algorithm Playground is a collection of interactive implementations and visualizations of common computer science algorithms.

The purpose of this project is to deepen my understanding of algorithms while creating an educational tool that demonstrates how they work.

## Planned Features

* Sorting algorithm visualizations
* Binary Search visualization
* Graph algorithms
* Pathfinding algorithms
* Time complexity comparisons
