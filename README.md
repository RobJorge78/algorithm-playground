# Algorithm Playground

## Overview

Algorithm Playground is an interactive desktop application built in Python that demonstrates how common computer science algorithms work through visualization and animation.

The goal of this project is to strengthen my understanding of algorithms while creating a learning tool for students and developers.

## Planned Features

### Searching

* Binary Search

### Sorting

* Bubble Sort
* Selection Sort
* Merge Sort
* Quick Sort

### Graph Algorithms

* Breadth-First Search (BFS)
* Depth-First Search (DFS)
* Dijkstra's Algorithm

## Technologies

* Python
* Tkinter
* Git
* GitHub

## Learning Goals

* Improve algorithm implementation skills
* Practice software architecture
* Build an interactive desktop application
* Write clean, maintainable code
